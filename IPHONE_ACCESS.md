# 📱 iPhone 云端访问指南

## 🚀 方法 1：GitHub Codespaces（推荐）

### 在 iPhone Safari 浏览器中：

1. **打开链接**：https://github.com/haitaibaobaobao-sketch/project-3.0-beishenyan
2. **点击绿色按钮** "Code"
3. **选择** "Codespaces" 标签
4. **点击** "Create codespace on master"
5. **等待 30 秒**，云端 VS Code 会自动打开
6. **在终端运行命令**（点击底部 Terminal）

**优点**：
- ✅ 完全免费（每月 60 小时）
- ✅ 自动保存所有更改
- ✅ 笔记本关机也能用
- ✅ 完整的 Linux 环境

---

## 🌐 方法 2：Replit（备用方案）

### 在 iPhone Safari 浏览器中：

1. **打开**：https://replit.com/
2. **登录**（用 GitHub 账号登录）
3. **点击** "Import from GitHub"
4. **粘贴仓库地址**：`https://github.com/haitaibaobaobao-sketch/project-3.0-beishenyan`
5. **点击** "Import"
6. **在 Shell 中运行命令**
